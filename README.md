# Portfolio Website

Welcome to my portfolio website! This project showcases my skills, experiences, and projects in a clean and professional manner.

## About the Project

This project is built using React and Next.js, inspired by the [react-resume-template](https://github.com/tbakerx/react-resume-template) by tbakerx. The goal of this website is to provide a comprehensive overview of my work and capabilities as a software developer.

## Features

- **Responsive Design**: The website is fully responsive, ensuring it looks great on all devices.
- **Dynamic Content**: Easily update and manage content using the power of React and Next.js.
- **Optimized Performance**: Leveraging Next.js for fast page loads and seamless navigation.
- **Modern UI**: A clean and modern user interface that highlights my professional background.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Next.js**: A React framework for server-side rendering and static site generation.
- **CSS Modules**: Scoped and modular CSS for styling components.
- **JavaScript**: The core programming language for the project.
